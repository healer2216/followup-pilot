#### FastAPI 服务
- ✅ RESTful API 架构

- ✅ **API Key**: 已配置并验证通过
- ✅ **Fallback 机制**: LLM 失败时自动降级到 Mock 数据

#### 核心服务模块

##### Parser Service (parser.py)
- ✅ LLM 驱动的出院小结解析
- ✅ JSON Mode 结构化输出
- ✅ 置信度计算
- ✅ 提取字段：患者信息、诊断、用药、生命体征

##### Evidence Service (evidence.py)
- ✅ KnowS API HTTP 客户端封装
- ✅ 支持 6 种证据源：paper_en, paper_cn, meeting, guide, trial, package_insert
- ✅ 串行查询策略（避免限流）
- ✅ 健康检查功能

##### Planner Service (planner.py)
- ✅ 融合患者数据 + 循证证据生成计划
- ✅ 时间线结构（第1天/第3天/第1周/第2周/第1月）
- ✅ 证据引用统计

##### Chatbot Service (chatbot.py)
- ✅ 对话随访引擎
- ✅ **AlertEngine 预警规则引擎**（红/黄/绿三级）
- ✅ 数值提取 + 关键词检测
- ✅ 按需证据检索
- ✅ 会话状态管理

##### Session Store (store.py)
- ✅ JSON 文件持久化存储
- ✅ 简单的 save/get/list_all 接口

## 🐛 已知问题与解决方案

### 问题1：预览工具无法显示页面

**原因**: 预览工具对 `localhost:8080` URL 有限制

**解决方案**: 
- ✅ 直接在浏览器访问 `http://localhost:8080`
- ✅ 使用 curl 测试 API
- ✅ 查看 [访问指南.md](file:///root/study/项目开发/followup-pilot/访问指南.md)

### 问题2：CDN 资源加载失败

**原因**: 网络环境限制或 CDN 不可用

**解决方案**: 
- ✅ 已创建 standalone.html（无 CDN 依赖）
- ✅ FastAPI 优先返回独立版

### 问题3：LLM API 调用失败

**原因**: API Key 错误或配额不足

**解决方案**: 
- ✅ 已配置 Fallback 机制（自动降级到 Mock）
- ✅ 检查 `.env` 文件中的 API Key
